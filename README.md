# Installation
- via WGET
```
bash -c "$(wget -qO- https://raw.githubusercontent.com/st4ngkudut/free-XRAY/main/install-xray)"
```
- via CURL
```
bash -c "$(curl -fsSL https://raw.githubusercontent.com//st4ngkudut/free-XRAY/main/install-xray)"
```

# NOTES
